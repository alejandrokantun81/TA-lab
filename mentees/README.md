# Mentee scratch spaces

Each mentee works in their own folder here: `mentees/<your-github-handle>/`.

- Create it on your first PR. Put your exercise work and experiments inside.
- Don't edit another mentee's folder.
- Suggested layout: `mentees/<handle>/sprint-01/`, `mentees/<handle>/sprint-02/`, …

This keeps everyone's work isolated and lets mentors see how each person works.
