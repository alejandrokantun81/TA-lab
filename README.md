# TA-lab

> Practice & experiments for **Talent Angels** — a low-bar sandbox to learn by
> doing before code lands in `TA-agents`.

Part of the [`LFX-Talent-Angels`](https://github.com/LFX-Talent-Angels) org. For
project-wide docs and rules, see
[`TA-workspace`](https://github.com/LFX-Talent-Angels/TA-workspace).

## What's here

- **`exercises/`** — briefs set by mentors. Start with
  [`sprint-01-taxonomy-graph/`](./exercises/sprint-01-taxonomy-graph).
- **`mentees/<your-handle>/`** — your own scratch space. Experiment freely.

This repo is **throwaway-friendly**: prototypes, notebooks, spikes, quick checks.
It is *not* production code — that lives in `TA-agents` with the full bar.

## How to use it

1. Create your folder: `mentees/<your-github-handle>/`.
2. Do the current exercise there (or any experiment).
3. Branch, `git commit -s` (DCO), open a PR. Review is light — it's for learning
   feedback and so mentors can see how you work, not for gatekeeping.

## Rules (few, but firm)

- DCO sign-off on every commit (`git commit -s`).
- No secrets. No personal data (nothing from the `Mentees/` folder).
- Work in your own folder; don't edit someone else's.
- Keep large data out of git (link to it or `.gitignore` it locally).
